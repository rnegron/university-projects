IEEE 754 floating point binary converter

This is the final project for my Computer Science 101 class.

We are assuming 32 bit precision for both conversions.

For more information on IEEE 754, please visit this page.

Examples of possible inputs: -42.1337 for conversion to binary and 11000010001010001000100011101000 for conversion to decimal. Rounded
